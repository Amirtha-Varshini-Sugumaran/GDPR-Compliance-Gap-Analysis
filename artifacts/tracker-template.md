## Suggested Status Values

- Open
- In progress
- Waiting on internal search
- Waiting on candidate response
- Closed

## Minimum Fields to Keep

- Request ID
- Date received
- Request type
- Owner
- Due date
- Status
- Evidence or completion notes

The point is not to create admin for the sake of it. The tracker gives the business one place to see what needs attention and what has already been completed.
