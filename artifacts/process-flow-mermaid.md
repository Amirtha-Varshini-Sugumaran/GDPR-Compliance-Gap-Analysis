# Process Flow

This shows the current manual GDPR request flow and the improved version proposed for Greenline Recruitment.

## As-Is Process

```mermaid
flowchart LR
    A[Candidate sends request by email] --> B[Shared inbox checked manually]
    B --> C[Someone searches emails and folders]
    C --> D[Response is sent]
    D --> E[No central record of closure]
```

## To-Be Process

```mermaid
flowchart LR
    A[Request received] --> B[Logged in GDPR tracker]
    B --> C[Owner assigned]
    C --> D[Data locations checked]
    D --> E[Action completed]
    E --> F[Response sent]
    F --> G[Evidence saved and request closed]
```

## Why This Helps

The improved flow gives the team a clear owner, a due date, and a record of what happened. It is still simple enough for a small business to manage.
